## 📖 Project Overview

This project focuses on building a **machine learning model** to classify iris flowers into three species:

* Setosa
* Versicolor
* Virginica

The model is trained using the famous **Iris dataset** and evaluated using accuracy and confusion matrix.

---

* Logistic Regression gives high accuracy on Iris dataset
* Setosa is classified perfectly
* Minor misclassification occurs between Versicolor & Virginica
